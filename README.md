🌿 EcoTrack - Monitor de Pegada de CarbonoO EcoTrack é uma aplicação web simples e intuitiva projetada para ajudar usuários a calcularem e monitorarem sua emissão estimada de $CO_2$ com base em hábitos diários (transporte, alimentação e consumo de energia).O projeto foi desenvolvido como atividade prática para a disciplina de Programação para Internet.🚀 FuncionalidadesCalculadora Dinâmica: Formulário que calcula a emissão de carbono em tempo real.Dicas Sustentáveis: Sugestões personalizadas baseadas no resultado do usuário.Histórico Local: Salva os últimos cálculos realizados utilizando localStorage.Modo Escuro (Dark Mode): Interface adaptável para maior conforto visual.🛠️ Tecnologias UtilizadasHTML5: Estrutura semântica para acessibilidade.CSS3: Layout moderno utilizando CSS Variables e Flexbox.JavaScript (Vanilla): Lógica de cálculo, manipulação do DOM e persistência de dados.Lucide Icons: Biblioteca de ícones leves para a interface.🧮 Lógica de CálculoA aplicação utiliza fatores de emissão simplificados para gerar o resultado. A fórmula base segue a estrutura:$$E_{total} = (D \cdot f_t) + (K \cdot f_e) + C_a$$Onde:$D$: Distância percorrida em km.$f_t$: Fator de emissão do transporte escolhido.$K$: Consumo de energia em kWh.$C_a$: Constante de emissão baseada no tipo de dieta.📦 Como TestarFaça o download ou clone este repositório.Navegue até a pasta do projeto.Abra o arquivo index.html em seu navegador de preferência.(Opcional) Utilize a extensão Live Server no VS Code para visualizar alterações em tempo real.📂 Estrutura de ArquivosPlaintext├── index.html         # Estrutura principal da página
